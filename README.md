# SPGCR : Suivi de Production et Gestion des Coûts de Revient

## 🚀 À propos du projet
SPGCR est une application web multisite conçue pour les PME industrielles de Goma (RDC). Elle permet de transformer une gestion artisanale en un système numérique intégré, offrant une visibilité en temps réel sur la transformation des matières premières et la rentabilité financière.

## 🛠 Stack Technique
- **Framework :** [Next.js 14+](https://nextjs.org/ ) (App Router)
- **Base de données & Auth :** [Supabase](https://supabase.com/ ) (PostgreSQL)
- **Styling :** [Tailwind CSS](https://tailwindcss.com/ ) + [Shadcn/UI](https://ui.shadcn.com/ )
- **Langage :** TypeScript

## ✨ Fonctionnalités Clés
- **Gestion Multisite :** Dashboard global pour le DG et vues locales pour les gérants.
- **Moteur de Production (BOM) :** Fiches techniques avec versioning et déstockage automatique.
- **Intelligence Financière :** Calcul automatique du coût de revient (charges directes et indirectes).
- **Audit Trail :** Journalisation inviolable de toutes les actions du système.
- **PWA :** Optimisé pour une utilisation mobile sur le terrain.
